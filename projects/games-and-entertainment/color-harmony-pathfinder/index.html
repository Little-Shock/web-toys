How to Play Color Harmony Pathfinder

Goal: Connect the Start Tile (glowing green) to the End Tile (glowing red) by creating a path of colored tiles.

Movement: You can only select tiles that are directly adjacent (not diagonally) to the last tile in your current path.

Color Harmony Rules: Each level has a specific color harmony rule. You must select tiles whose color harmonizes with the previous tile according to the current level's rule (e.g., "Shades", "Analogous", "Complementary").

Starting: Click a level button to load it. Then, click the designated Start Tile to begin your path.

Feedback:

  • Valid next moves may be highlighted when you hover over them.
  • If you make an invalid move, the path will reset.
